react-tutorial-with-backend
===========================

This is a version of the [React tutorial](http://facebook.github.io/react/docs/tutorial.html) with a small backend written in Ruby.

## How to use it

```
$ bundle
$ ruby app.rb
# go to http://localhost:4567
```
